2. ¿Qué métodos tiene el objeto Date?

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.getTime();
</script>

Método Descripción
getDate() Obtiene el día como númuero (1-31)
getDay() Obtiene el día de la semanacomo número (0-6)
getFullYear() Obtiene el año actual (yyyy)
getHours() Obtiene la hora actual (0-23)
getMilliseconds() Obtiene los milisegundos actuales (0-999)
getMinutes() Obtiene el minuto actual (0-59)
getMonth() Obtiene el mes actual (0-11)
getSeconds() Obtiene los segundos actuales (0-59)
getTime() Obtiene el tiempo transcurrido desde Enero 1 1970

3. ¿Qué métodos tienen los arreglos?

var fruits = ["index0", "index1", "index2", "index3"];
document.getElementById("demo").innerHTML = fruits.toString();

Método Descripción
.toString() Convierte un arreglo a un string de valores de arreglos separados por una coma.
.join() Une los elementos del arreglo en un string. Además puedes especificar el separador. Ejemplo: fruits.join(" * ");
.pop() Elimina el último elemento del arreglo
.push() Añade un elemento al arreglo en la última posición.
.shift() Elimina el primer elemento del arreglo y desplaza todos una posición a la izquierda. Regresa el elemento eliminado.
.unshift() Añade un elemento al inicio del array y recorre todos a la derecha. Regresa el numero de elementos.
.concat() Concatena dos arreglos. Ejemplo: arreglo3= arreglo1.concat(arreglo2)

Las expresiones regulares son patrones utilizados para encontrar una determinada combinación de caracteres dentro de una cadena de texto. En JavaScript, las expresiones regulares también son objetos. [4]
Los patrones de string que hacen uso de las expresiones regulares son:

  • Match
  • Replace
  • Search
  • Split
